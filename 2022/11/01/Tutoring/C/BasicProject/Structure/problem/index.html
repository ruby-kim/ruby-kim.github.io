🅒 C언어 기초: 구조체 (3) 추가 연습문제

  1. 5명의 학생의 학번과 국어, 영어, 수학 점수를 저장할 수 있는 구조체를 만들고, 각 학생의 평균 점수(소수점 둘째 자리 까지)를 출력하는 함수를 작성해주세요.

    /* 구조체 선언 */
typedef struct {
    // 국어, 영어, 수학, 학번 선언
} score;

/* 사용해야 하는 함수 */
double avg(score *st);

  1. 공부 시작시간과 종료시간을 입력받아 소요시간을 구하려고 합니다.
    두 개의 시(0-24), 분(0-60), 초(0-60)를 각각 입력받아 계산해주세요.

    /* 구조체 선언 */
typedef struct {
    // 멤버는 잘 선언해주기!
} Time;

/* 사용해야 하는 함수: 시간차 계산 함수 */
// Time A: 시작시간, Time B: 종료시간, Time *d: 시간차
void time_diff(Time A, Time B, Time *d)

    // 입력
시작시간(시, 분, 초): 1 10 23
죵료시간(시, 분, 초): 3 15 43

// 출력
소요시간: 2 50 20

    // 입력
시작시간(시, 분, 초): 1 10 23
죵료시간(시, 분, 초): 3 5 23

// 출력
소요시간: 1 55 0

  1. 복소수의 덧셈, 뺄셈 계산기를 만들어주세요. 숫자(a, b)는 모두 정수로 이루어져 있습니다.

    /* 구조체 선언 */
typedef struct {
    // 멤버는 잘 선언해주기!
} Complex_num;


/* 사용해야 하는 함수: 복소수 입력 (a, b 저장) */
void get_input(Complex_num *p)

/* 사용해야 하는 함수: 복소수 덧셈 계산기 */
Complex_num calc_add(Complex_num A, Complex_num B)

/* 사용해야 하는 함수: 복소수 뺄셈 계산기 */
Complex_num calc_substract(Complex_num A, Complex_num B)

    // 입력
첫 번째 복소수(a + bi)를 입력해주세요 (a, b): 3 5
두 번째 복소수(a + bi)를 입력해주세요 (a, b): 2 7

// 출력
합계: 5+12i
뺄셈: 1-2i

    // 입력
첫 번째 복소수(a + bi)를 입력해주세요 (a, b): 3 5
두 번째 복소수(a + bi)를 입력해주세요 (a, b): 3 5

// 출력
합계: 6+10i
뺄셈: 0
